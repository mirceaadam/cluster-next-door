## Key Design Principles

- **No port forwarding** вҖ” all public ingress via Cloudflare Tunnels
- **Zero cloud lock-in** вҖ” runs on recycled hardware, cloud only for DR
- **Idempotent infrastructure** вҖ” everything reproducible via Terraform + Ansible
- **Survive single-site failure** вҖ” workloads on both houses, DNS failover ready
- **Solar powered** вҖ” both sites run on renewable energy
